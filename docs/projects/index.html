Explore with Shiny: Creating a Statistics Inquiry-Based Learning Tool

Creating a dynamic interactive shiny application in R to better students’ understanding of statistics and data science. The application focuses on bootstrapping and permutations. Our application provides students the opportunity to learn about these concepts through investigation, while simultaneously developing the needed coding skills to perform these concepts on their own.

This project aims to support teaching at Duke and other universities by continuing the advancement of statistics and data science education, specifically at the undergraduate level. Further, this application sets the foundation for future work to best understand how students learn to code and implement these processes when first being introduced to bootstrapping and permutations.

This project plans to extend to include the development of assessment questions surrounding the application.

Creating a Data Science Specialization in Coursera

Creating a multi-course Data Science with R specialization for Coursera. This includes modules such as: Data Visualization; Importing + Wrangling Data; Data Ethics; Modeling + Prediction; Inference. This project entails the creation of videos and assessment questions on such topics that self-enrolled students must watch and complete before earning their Data Science with R specialization.

Understanding how self-determined their use of active learning is through their motivations to use active learning can help inform how to support newer instructors’ adoption of such techniques.
